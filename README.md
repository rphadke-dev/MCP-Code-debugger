# Simulation Engine + AI Debugger (MCP)

A Python-based Model Context Protocol (MCP) server that exposes
simulation and debugging tools to Claude Desktop.

## Features
- MCP server implemented in Python
- Tool-based interface for simulations
- Designed for AI-assisted debugging and analysis
- Runs locally and integrates with Claude Desktop

## Requirements
- Python 3.10+
- Claude Desktop
- Windows / macOS / Linux

## Setup

```bash
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
pip install -r requirements.txt
